## 説明 {#description}

`Null`フォーマットでは、何も出力されません。 
最初は奇妙に感じるかもしれませんが、何も出力されないにも関わらず、クエリは処理されることを理解することが重要です。 
コマンドラインクライアントを使用する際には、データがクライアントに送信されます。 

:::tip
`Null`フォーマットはパフォーマンステストに役立つことがあります。
:::
