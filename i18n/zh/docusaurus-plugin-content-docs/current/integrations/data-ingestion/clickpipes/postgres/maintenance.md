# Postgres ClickPipes 的维护窗口

近期将有一个 Postgres ClickPipes 的维护窗口，定于：
- **日期:** 2025年4月17日
- **时间:** UTC 时间 07:00 AM - 08:00 AM

在此期间，您的 Postgres Pipes 将会经历短暂的停机时间。
维护窗口结束后，ClickPipes 将重新可用，并恢复正常操作。
