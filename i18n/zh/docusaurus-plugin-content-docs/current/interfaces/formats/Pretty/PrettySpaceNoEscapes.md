## Description {#description}

与 [`PrettySpace`](./PrettySpace.md) 格式不同的是，不使用 [ANSI-escape sequences](http://en.wikipedia.org/wiki/ANSI_escape_code)。 
这是在浏览器中显示该格式，以及使用 'watch' 命令行工具所必需的。
