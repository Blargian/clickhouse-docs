Интеграции ClickHouse организованы по уровню их поддержки:

- **Основные интеграции:** разработанные или поддерживаемые ClickHouse, они поддерживаются ClickHouse и находятся в организации ClickHouse на GitHub
- **Интеграции партнеров:** разработанные или поддерживаемые сторонними поставщиками программного обеспечения
- **Интеграции сообщества:** разработанные или поддерживаемые членами сообщества. Прямой поддержки не предоставляется, кроме публичных репозиториев на GitHub и каналов Slack сообщества

Каждая интеграция дополнительно классифицируется по категориям **Языковой клиент**, **Прием данных**, **Визуализация данных** и **SQL клиент**.

:::note
Мы активно составляем этот список интеграций ClickHouse ниже, поэтому он не исчерпывающий. Не стесняйтесь 
[внести свой вклад](https://github.com/ClickHouse/clickhouse-docs#contributing) в список с любой соответствующей интеграцией ClickHouse.
:::
