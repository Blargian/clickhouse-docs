# Postgres用のClickPipesのメンテナンスウィンドウ

Postgres ClickPipesのメンテナンスウィンドウが以下の日程で予定されています：
- **日付:** 2025年4月17日
- **時間:** 07:00 AM - 08:00 AM UTC

この期間中、あなたのPostgres Pipesは一時的にダウンタイムを経験します。
メンテナンスウィンドウ後、ClickPipesは再び利用可能になり、通常の操作を再開します。
