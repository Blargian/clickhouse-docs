## 描述 {#description}

与 [`JSONCompactEachRow`](./JSONCompactEachRow.md) 的区别在于数据字段作为字符串输出，而不是作为类型化的 JSON 值。

## 示例用法 {#example-usage}

示例：

```json
["42", "hello", "[0,1]"]
["43", "hello", "[0,1,2]"]
["44", "hello", "[0,1,2,3]"]
```
