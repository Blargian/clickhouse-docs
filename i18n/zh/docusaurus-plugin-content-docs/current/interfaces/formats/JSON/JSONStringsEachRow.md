## 描述 {#description}

与 [`JSONEachRow`](./JSONEachRow.md) 的唯一不同在于，数据字段以字符串形式输出，而不是以类型化的 JSON 值。

## 示例用法 {#example-usage}

```json
{"num":"42","str":"hello","arr":"[0,1]"}
{"num":"43","str":"hello","arr":"[0,1,2]"}
{"num":"44","str":"hello","arr":"[0,1,2,3]"}
```
