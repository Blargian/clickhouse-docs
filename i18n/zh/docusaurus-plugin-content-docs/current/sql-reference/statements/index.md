# ClickHouse SQL 语句

用户使用 SQL 语句与 ClickHouse 进行交互。ClickHouse 支持常见的 SQL 语句，如 [SELECT](select/index.md) 和 [CREATE](create/index.md)，但它还提供了专门的语句，如 [KILL](kill.md) 和 [OPTIMIZE](optimize.md)。
