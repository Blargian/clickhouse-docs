ClickHouse Cloudは、転送されるデータの入出力を計測します。 
これには、ClickHouse Cloudへのデータの出入り、および地域内データ転送と地域間データ転送が含まれます。 
この使用量はサービスレベルで追跡されます。この使用に基づき、顧客はデータ転送料金が発生し、それが月額請求書に追加されます。

ClickHouse Cloudでは以下のデータ転送に対して料金が発生します:
- ClickHouse Cloudからパブリックインターネットへのデータの出力（他のクラウドプロバイダーの他の地域へも含む）。
- 同じクラウドプロバイダー内の別の地域へのデータの出力。

地域内データ転送やプライベートリンク/プライベートサービス接続の使用およびデータ転送には料金は発生しません。 
ただし、ユーザーに適切に課金する能力に影響を与える使用パターンが見られた場合、追加のデータ転送料金の次元を実装する権利を留保します。

データ転送料金はクラウドサービスプロバイダー（CSP）および地域によって異なります。 
パブリックインターネットへの出力料金は、発信地域のみに基づいています。 
地域間（またはクロスリージョン）の料金は、発信地域と宛先地域の両方に依存します。
