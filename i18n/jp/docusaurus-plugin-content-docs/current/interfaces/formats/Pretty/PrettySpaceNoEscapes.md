## 説明 {#description}

[`PrettySpace`](./PrettySpace.md)フォーマットとは異なり、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code)は使用されていません。  
これはこのフォーマットをブラウザで表示する際、及び 'watch' コマンドラインユーティリティを使用するために必要です。
