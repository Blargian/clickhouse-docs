# avgMap

The [`Map`](/sql-reference/aggregate-functions/combinators#-map) combinator can be applied to the [`avg`](/sql-reference/aggregate-functions/reference/avg) function to calculate the average of values for each key in a map using the `avgMap` function.

This is useful when you want to calculate average values by key in a map structure.
