ClickHouseの統合は、サポートレベルに応じて整理されています：

- **コア統合：** ClickHouseによって構築または管理され、ClickHouseにサポートされているもので、ClickHouseのGitHub組織に存在します。
- **パートナー統合：** サードパーティのソフトウェアベンダーによって構築または管理され、サポートされているものです。
- **コミュニティ統合：** コミュニティメンバーによって構築または管理されたもので、サポートは公開GitHubリポジトリやコミュニティのSlackチャンネルに限られます。

各統合は、**言語クライアント**、**データ取り込み**、**データ可視化**、**SQLクライアント**のカテゴリにさらに分類されます。

:::note
私たちは、以下のClickHouse統合のリストを積極的に編集していますので、網羅的ではありません。関連するClickHouse統合があれば、ぜひ
[貢献](https://github.com/ClickHouse/clickhouse-docs#contributing)してください。
:::
