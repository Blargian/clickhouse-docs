ClickHouseとのインテグレーションは、そのサポートレベルによって整理されています：

- **コアインテグレーション:** ClickHouseによって構築または維持され、ClickHouseによってサポートされており、ClickHouseのGitHub組織に存在します。
- **パートナーインテグレーション:** 第三者ソフトウェアベンダーによって構築または維持され、サポートされています。
- **コミュニティインテグレーション:** コミュニティメンバーによって構築または維持され、サポートされています。公のGitHubリポジトリとコミュニティSlackチャンネルを除いて、直接のサポートは利用できません。

各インテグレーションは、**言語クライアント**、**データ取り込み**、**データ可視化**、**SQLクライアント**のカテゴリにさらに分類されます。

:::note
以下にClickHouseのインテグレーションリストを積極的に編纂していますので、網羅的ではありません。関連するClickHouseインテグレーションをリストに[寄与](https://github.com/ClickHouse/clickhouse-docs#contributing)することを自由に行ってください。
:::
