# Postgres ClickPipes 的维护窗口

即将到来的 Postgres ClickPipes 维护窗口定于：
- **日期：** 2025年4月17日
- **时间：** UTC 时间 07:00 - 08:00

在此期间，您的 Postgres Pipes 将会经历短暂的停机。
维护窗口结束后，ClickPipes 将会重新可用并恢复正常操作。
