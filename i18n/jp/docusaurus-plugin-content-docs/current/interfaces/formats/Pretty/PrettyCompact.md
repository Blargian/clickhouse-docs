## 説明 {#description}

[`Pretty`](./Pretty.md)フォーマットとは異なり、行の間にグリッドが描かれたテーブルとして表示されます。 
そのため、結果はよりコンパクトになります。

:::note
このフォーマットは、インタラクティブモードのコマンドラインクライアントでデフォルトで使用されます。
:::
