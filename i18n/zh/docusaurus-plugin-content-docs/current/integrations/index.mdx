ClickHouse 集成按其支持级别进行组织：

- **核心集成：** 由 ClickHouse 构建或维护，得到 ClickHouse 的支持，并位于 ClickHouse GitHub 组织中
- **合作伙伴集成：** 由第三方软件供应商构建或维护并支持
- **社区集成：** 由社区成员构建或维护并支持。除了公共 GitHub 存储库和社区 Slack 频道外，没有直接支持可用

每个集成都进一步分类为 **语言客户端**、**数据摄取**、**数据可视化** 和 **SQL 客户端** 类别。

:::note
我们正在积极整理以下 ClickHouse 集成列表，因此它并不详尽。欢迎您
[贡献](https://github.com/ClickHouse/clickhouse-docs#contributing) 任何相关的 ClickHouse 集成到列表中。
:::
