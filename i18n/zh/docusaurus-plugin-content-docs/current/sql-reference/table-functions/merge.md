
# merge 表函数

创建一个临时的 [Merge](../../engines/table-engines/special/merge.md) 表。其结构将通过对底层表的列进行联合，并推导出公共类型来生成。

## 语法 {#syntax}

```sql
merge(['db_name',] 'tables_regexp')
```
## 参数 {#arguments}

| 参数            | 描述                                                                                                                                                                                                                                      |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `db_name`       | 可选值（默认值为 `currentDatabase()`）：<br/>    - 数据库名称,<br/>    - 返回数据库名称字符串的常量表达式，例如 `currentDatabase()` ,<br/>    - `REGEXP(expression)`，其中 `expression` 是用于匹配数据库名称的正则表达式。  |
| `tables_regexp` | 用于匹配指定数据库或数据库中的表名称的正则表达式。                                                                                                                                                                                                                       |
