# 视图表引擎

用于实现视图（有关更多信息，请参见 `CREATE VIEW 查询`）。它不存储数据，而仅存储指定的 `SELECT` 查询。当从表中读取时，它会运行此查询（并从查询中删除所有不必要的列）。
