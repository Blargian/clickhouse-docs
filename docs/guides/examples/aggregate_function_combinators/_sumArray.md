# sumArray

The [`Array`](/sql-reference/aggregate-functions/combinators#-array) combinator can be applied to the [`sum`](/sql-reference/aggregate-functions/reference/sum) function to 
calculate the sum of all elements in an array using the `sumArray` function.

This is useful when you want to sum all elements in an array without having to 
use `arraySum` or `arrayJoin`.
