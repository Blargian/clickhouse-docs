| 页面                                                          | 描述                                                                                                                                                                                              |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [共享责任模型](shared-responsibility-model.md)               | 关于每种服务类型提供的安全功能的信息。                                                                                                                                                             |
| [云访问管理](cloud-access-management/index.md)                | 访问控制、身份验证、SSO 设置、常见访问管理查询及如何邀请新用户的信息。                                                                                                                            |
