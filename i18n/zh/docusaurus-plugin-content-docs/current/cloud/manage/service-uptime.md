## Uptime Alerts {#uptime-alerts}

用户现在可以在 [status page](https://status.clickhouse.com/) 上查看区域正常运行时间，并订阅服务中断的提醒。

我们还为部分承诺支出合同提供服务水平协议（SLA）。请通过 [sales@clickhouse.com](mailto:sales@clickhouse.com) 联系我们以了解更多关于我们 SLA 政策的信息。
