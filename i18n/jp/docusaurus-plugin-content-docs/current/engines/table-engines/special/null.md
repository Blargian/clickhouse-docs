# Null テーブルエンジン

`Null` テーブルに書き込むと、データは無視されます。`Null` テーブルから読み込むと、レスポンスは空になります。

:::note
これがなぜ有用か疑問に思うかもしれませんが、`Null` テーブルにマテリアライズドビューを作成できることに注意してください。したがって、テーブルに書き込まれたデータはビューに影響を与えることになりますが、元の生データは依然として破棄されます。
:::
