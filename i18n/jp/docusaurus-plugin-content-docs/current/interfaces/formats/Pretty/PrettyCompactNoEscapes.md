## 説明 {#description}

[`PrettyCompact`](./PrettyCompact.md) 形式と異なり、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code) は使用されません。 
これは、ブラウザで形式を表示するため、及び 'watch' コマンドラインユーティリティを使用するために必要です。
