# 集約関数

ClickHouseは、すべての標準SQL集約関数（[sum](../reference/sum.md)、[avg](../reference/avg.md)、[min](../reference/min.md)、[max](../reference/max.md)、[count](../reference/count.md)）に加え、幅広いその他の集約関数をサポートしています。
