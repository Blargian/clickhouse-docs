## 説明 {#description}

[`PrettySpace`](./PrettySpace.md) 形式とは異なり、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code) は使用されません。 
これは、この形式をブラウザで表示するため、および 'watch' コマンドラインユーティリティを使用するために必要です。
