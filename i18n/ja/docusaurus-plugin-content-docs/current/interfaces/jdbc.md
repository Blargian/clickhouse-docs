# JDBC ドライバー

[公式の JDBC ドライバー](https://github.com/ClickHouse/clickhouse-jdbc)（および Java クライアント）を使用して、Java アプリケーションから ClickHouse にアクセスします。

- サードパーティのドライバー：
    - [ClickHouse-Native-JDBC](https://github.com/housepower/ClickHouse-Native-JDBC)
    - [clickhouse4j](https://github.com/blynkkk/clickhouse4j)
