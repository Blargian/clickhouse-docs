## 説明 {#description}

[`Pretty`](./Pretty.md) 形式とは異なり、行の間にグリッドが描かれた状態でテーブルが表示されます。  
そのため、結果はよりコンパクトになります。

:::note
この形式は、対話モードでコマンドラインクライアントのデフォルトで使用されます。
:::
