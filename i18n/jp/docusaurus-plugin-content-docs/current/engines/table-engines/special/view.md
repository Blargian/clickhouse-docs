# View Table Engine

ビューを実装するために使用されます（詳細については、`CREATE VIEW クエリ`を参照してください）。データは保存せず、指定された `SELECT` クエリのみを保存します。テーブルから読み取る際には、このクエリを実行し（クエリからすべての不要なカラムを削除します）。
