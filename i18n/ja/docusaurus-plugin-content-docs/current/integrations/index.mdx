ClickHouseの統合はそのサポートレベルによって整理されています：

- **コア統合:** ClickHouseによって構築または維持され、ClickHouseによってサポートされ、ClickHouse GitHub組織に存在します。
- **パートナー統合:** 第三者のソフトウェアベンダーによって構築または維持され、サポートされています。
- **コミュニティ統合:** コミュニティメンバーによって構築または維持され、サポートされています。公開のGitHubリポジトリとコミュニティSlackチャネル以外の直接的なサポートはありません。

各統合はさらに**言語クライアント**、**データ取り込み**、**データ可視化**、および**SQLクライアント**のカテゴリに分類されます。

:::note
私たちは現在、以下のClickHouse統合のリストを積極的に編纂しているため、完全ではありません。関連するClickHouse統合をリストに
[寄与](https://github.com/ClickHouse/clickhouse-docs#contributing)することを気軽にどうぞ。
:::
